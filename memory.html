🃏 Port Memory

A classic concentration game — flip two cards, pair each port number with its protocol name only.
How to play
  1. All cards start face down. Click a card to flip it over.
  2. Click a second card. If one shows a port number and the other shows the matching protocol name, they stay face up.
  3. If they don't match, both flip back down after a moment — remember what you saw.
  4. Clear the whole board in as few flips as possible.
